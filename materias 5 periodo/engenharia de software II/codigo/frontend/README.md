## How to run
```bash
# Clone Repository
$ git clone https://github.com/CDGMV/frontend.git
```
### Run Web Project

```bash
# Go to web folder
$ cd frontend

# Install Dependencies
$ npm install

# Run Aplication
$ npm start
```
Go to http://localhost:3000/ to see the result.
